# Overview

Testing is a process that is essential to the quality of any product. By understanding the seven steps of the testing process, Test Engineers can have a working model for how to test products and ensure their quality. The seven steps of the testing process are Test Planning, Test Monitoring and Control, Test Analysis, Test Design, Test Implementation, Test Execution, and Test Completion. Let's review each step in more detail.

# The Seven Steps

## Test Planning

Test planning answers questions such as, "Who will do testing?" or "What test activities will we use? The activity involves determining the objectives of the testing effort, developing a strategy for achieving those objectives, and creating a schedule and budget for the tests.

## Test Monitoring and Control

Test monitoring answers questions such as, "How are we doing so far?" or "Are we on track to complete test plan objectives?". Test Monitoring and Control ensures that tests proceed as planned and that any deviations from the plan are identified and corrected.

## Test Analysis

Test analysis answers the question, "What to test?" and involves understanding the system's requirements under test and developing a plan for how you will test those requirements.

## Test Design

Test design answers the question, "How to test?" and involves creating detailed plans for what tests will be conducted and how you will execute them.

## Test Implementation

Test implementation answers the question, "Do we now have everything in place to run the tests?" and involves conducting the tests according to the plans developed in the previous steps.

## Test Execution

Test execution is the process of running the tests and recording the results.

## Test Completion

Test completion answers the question, "What do we do now that testing is done?" and involves analyzing the test results, writing a report on the findings, and making recommendations for future action.
